# 📊 Customer Offer Engagement Dashboard – Power BI Project

## 🔎 Project Overview

This Power BI dashboard analyzes customer engagement with marketing offers.
It helps businesses understand offer performance, customer participation trends, and transaction behavior to improve marketing effectiveness and revenue generation.

---

## 🎯 Business Objective

The goal of this project is to:

* Track number of offers sent vs completed
* Measure offer redemption rate
* Analyze customer transaction patterns
* Identify engagement trends over time
* Support data-driven marketing decisions

---

## 📁 Dataset Information

* Customer transaction and marketing offer dataset
* Includes customer demographics, offer interactions, and transaction events
* Data cleaned and transformed using Power Query

---

## 🛠 Tools & Technologies Used

* Power BI Desktop
* Power Query
* DAX (Data Analysis Expressions)
* Data Modelling
* Interactive Dashboard Design

---

## 📌 Key Dashboard Features

### ✅ KPI Cards

* Total Customers
* Offers Sent
* Offers Completed
* Redemption Rate
* Average Transaction Amount

### ✅ Offer Performance Analysis

* Comparison of offers sent vs offers completed
* Helps identify campaign effectiveness

### ✅ Event Distribution

* Breakdown of customer activities:

  * Offer Viewed
  * Offer Received
  * Offer Completed
  * Transactions

### ✅ Time Trend Analysis

* Daily offer engagement trends
* Transaction patterns over 30 days

### ✅ Customer Segmentation

* Membership year-wise new member analysis
* Gender-based customer insights

---

## 📈 Key Insights

* Not all sent offers are converted into completed offers
* Certain event types dominate customer engagement
* Customer participation varies over time
* Redemption rate provides strong indicator of campaign success

---

## 🚀 Business Impact

This dashboard can help marketing teams to:

* Improve offer targeting strategy
* Optimize campaign timing
* Increase customer conversion rates
* Enhance revenue through better engagement
